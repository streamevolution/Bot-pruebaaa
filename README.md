# Revolution Botcito

Bot básico para WhatsApp con bienvenida personalizada y comandos para asesor de ventas.

## Características

- Mensaje de bienvenida la primera vez que un usuario escribe.
- Comandos: `/productos`, `/ofertas`, `/contacto`.
- Envio de fotos y videos asociados a los comandos.
- Código en Node.js usando Baileys.
